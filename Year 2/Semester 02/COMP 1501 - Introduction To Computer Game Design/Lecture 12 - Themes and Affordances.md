
# Context Through Theme

Thematic elements make the game easier to learn and understand.
Themes help to set expectations for play.

**Example :**
Fallout: New Vegas
	Post-Nuclear Nevada Desert  
	What enemies do you expect to face?  
	What are weapons like?
